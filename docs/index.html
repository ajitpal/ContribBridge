Three commands.
Zero friction.

No config files. No YAML hell. No language expertise needed. ContribBridge wires into your existing GitHub workflow in under a minute.

01
📦
Install & Init
One npx command scaffolds the full project — Lingo.dev SDK, Express webhook server, WebSocket dashboard, SQLite cache, and .env template.
# Scaffold everything in one shot npx contribbridge@latest init ✓ lingo.dev/sdk installed ✓ @octokit/rest installed ✓ Webhook secret generated ✓ Dashboard at :4000
02
🔑
Set Two Keys
Only two env vars required. The init wizard prompts for both. Public repos are free forever — add a license key only for private repos.
# .env — two required keys LINGODOTDEV_API_KEY=lingo_xxx GITHUB_TOKEN=ghp_xxx # Private repos (Pro only): CONTRIBBRIDGE_LICENSE_KEY=jwt_xxx
03
🚀
Connect & Watch
Connect any repo — auto-registers the GitHub webhook via Octokit. Every new issue flows through the Lingo.dev pipeline in under 600ms.
contribbridge connect --repo org/repo contribbridge watch → detectLocale() ~40ms → localizeHtml() ~300ms → postComment() ~120ms ✓ Total: <600ms
